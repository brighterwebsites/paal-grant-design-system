Render a Paal Grant label, badge, or tag element.

```jsx
<Badge variant="section-label">Selected Work</Badge>
<Badge variant="award">Award-Winning Landscape Design</Badge>
<Badge variant="size">Medium · Most Popular</Badge>
<Badge variant="tag">Onsite or Virtual · $350 credited</Badge>
<Badge variant="gold-title">International Award</Badge>
<Badge variant="size-badge">Small Package</Badge>
```

**Variants:**
- `section-label` — micro overline above section headings; sage-90, ultra letter-spacing.
- `award` — horizontal pill with sage border used above hero/intro headings.
- `size` — package tier label (Small / Medium / Large); inline, no border.
- `tag` — bordered pill for process step callouts.
- `breadcrumb` — page path indicator; muted color, uppercase.
- `gold-title` — award plaque category label in gold.
- `size-badge` — bordered package size badge on package detail page.
