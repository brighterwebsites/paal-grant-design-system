# Paal Grant Website — UI Kit

Full-page interactive website prototype for the Paal Grant Designs In Landscaping website.

## Pages
- **Home** — Hero slideshow, intro, featured projects, package cards overview, footer
- **Process** — Step-by-step 4-phase process
- **About** — Founder portrait, credentials, media awards
- **Design Packages** — Three-column package cards with deliverables
- **Package Detail** — Individual package gallery + detail view
- **Portfolio** — Full-width image grid
- **Contact** — Two-column contact form + info

## Usage
Open `index.html` to see the full interactive prototype. All pages are single-file SPA via `showPage()` JS routing.

## Design notes
- All type is Raleway (Google Fonts) — weights 100, 200, 300, 400
- Accent color: `#8a9b7a` (sage green) at various opacities
- Base background: `#0a0d08` (near-black with green undertone)
- Section alternates: `#242422`
- Border radius: 8px universal, 6px small controls
- See `../../readme.md` for full visual foundations documentation
