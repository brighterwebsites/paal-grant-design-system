Text label and badge component for Paal Grant — five variants covering all label/badge needs.

```jsx
<Badge variant="section-label">Selected Work</Badge>
<Badge variant="award">Award-Winning Landscape Design & Construction</Badge>
<Badge variant="size">Small</Badge>
<Badge variant="size">Medium · Most Popular</Badge>
<Badge variant="tag">Onsite or Virtual · $350 credited</Badge>
<Badge variant="breadcrumb">Design Packages / Small</Badge>
```

Notable variants:
- `section-label` — ultra-spaced sage overline, always above a section title
- `award` — bordered sage pill with icon slot, used in home-intro
- `size` — tiny caps, used on package card tops
- `tag` — bordered sage pill for process step callouts
- `breadcrumb` — muted caps for page-detail navigation context
